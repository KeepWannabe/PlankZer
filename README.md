# PlankZer [Plank Zeparator]
Simple script for adding spacers into Plank dock.

![Plank spacers](screenshot.png)

## Usage

    git clone git://github.com/bipolarizm/PlankZer.git ~/PlankZer
    cd ~/PlankZer
    chmod +x plankzer.sh && ./plankzer.sh
